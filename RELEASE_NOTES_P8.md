# 更新说明 · v0.12.0-p8

> 本次更新围绕"桌面小组件"和"日常使用可靠性"。升级后不用重填配置，老笔记老日程都在。

---

## 新功能

- **桌面小组件从"今天 3 条"升级为"最近 20 条可滚动"**：把小组件拉大到 4×4，一屏能看更多。
- **笔记变动立刻反映到桌面**：新写、删除、置顶、后台同步拉到新内容、切换 GitHub 配置，桌面都会自动跟着更新。
- **顶部多了手动刷新按钮**：遇到网络断续时，点一下 🔄 就能重新拉最新数据。
- **常见动作有即时提示**：保存 / 刷新等操作会弹一条轻量提示条，让你知道进展。
- **可以在 app 内删除笔记了**：之前只能去 GitHub 网页删，现在长按笔记就能删。

## 修复

- **笔记保存"只写一半"的极小概率问题已修复**：内容写入现在用完整事务保护，中途异常不会留下残缺记录。
- **密钥泄漏风险收紧**：GitHub PAT 和 AI API Key 不会被带进任何调试字符串 / 错误文本。
- **编辑器按返回键更安全**：不会把写到一半的内容悄悄丢掉。
- **启动性能更稳**：首次冷启动对数据库和网络组件做了精简，打开更快。

## 改进

- **启动首帧更干净**：打开那一刻不再阻塞等待后台任务。
- **切换底部 Tab 后滚动位置会保留**：去看一眼日历再切回来，还在原位。
- **后台同步的错误状态更诚实**：以前极小概率把"部分失败"显示成"全部成功"，现在会如实提示。

## 已知限制

以下几项暂未完成，会在下一个小版本（P8.1）修：

- **文案国际化 / 无障碍适配**：目前以中文硬编码为主，TalkBack 描述有改进空间。
- **视觉细节打磨**：深色模式对比度、按钮反馈、空列表插画等。
- **测试覆盖面继续扩大**：核心路径已有测试，边角场景（并发、特殊文件名、时区临界）会补齐。
- **内部代码结构整理**：不影响用户体验，是为后续迭代铺路。
